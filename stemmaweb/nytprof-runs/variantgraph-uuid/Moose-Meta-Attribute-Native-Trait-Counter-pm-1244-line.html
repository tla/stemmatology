61=head1 DESCRIPTION
62
63This trait provides native delegation methods for counters. A counter can be
64any sort of number (integer or not). The delegation methods allow you to
65increment, decrement, or reset the value.
66
67=head1 DEFAULT TYPE
68
69If you don't provide an C<isa> value for your attribute, it will default to
70C<Num>.
71
72=head1 PROVIDED METHODS
73
74=over 4
75
76=item * B<set($value)>
77
78Sets the counter to the specified value and returns the new value.
79
80This method requires a single argument.
81
82=item * B<inc>
83
84=item * B<inc($arg)>
85
86Increases the attribute value by the amount of the argument, or by 1 if no
87argument is given. This method returns the new value.
88
89This method accepts a single argument.
90
91=item * B<dec>
92
93=item * B<dec($arg)>
94
95Decreases the attribute value by the amount of the argument, or by 1 if no
96argument is given. This method returns the new value.
97
98This method accepts a single argument.
99
100=item * B<reset>
101
102Resets the value stored in this slot to its default value, and returns the new
103value.
