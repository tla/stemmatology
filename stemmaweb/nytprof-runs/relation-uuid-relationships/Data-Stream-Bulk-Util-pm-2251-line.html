67=head1 DESCRIPTION
68
69This module exports convenience functions for use with L<Data::Stream::Bulk>.
70
71=head1 EXPORTS
72
73L<Sub::Exporter> is used to create the C<import> routine, and all of its
74aliasing/currying goodness is of course supported.
75
76=over 4
77
78=item nil
79
80Creates a new L<Data::Stream::Bulk::Nil> object.
81
82Takes no arguments.
83
84=item bulk @items
85
86Creates a new L<Data::Stream::Bulk::Array> wrapping C<@items>.
87
88=item cat @streams
89
90Concatenate several streams together.
91
92Returns C<nil> if no arguments are provided.
93
94=item filter { ... } $stream
95
96Calls C<filter> on $stream with the provided filter.
97
98=item unique $stream
99
100Filter the stream to remove duplicates.
101
102Note that memory use may potentially scale to O(k) where k is the number of
103distinct items, because this is implemented in terms of a seen hash.
104
105In the future this will be optimized to be iterative for sorted streams.
106
107References are keyed by their refaddr (see L<Hash::Util::FieldHash/id>).
108
