38 has 'options' => (

61=head1 DESCRIPTION
62
63This trait provides native delegation methods for array references.
64
65=head1 DEFAULT TYPE
66
67If you don't provide an C<isa> value for your attribute, it will default to
68C<ArrayRef>.
69
70=head1 PROVIDED METHODS
71
72=over 4
73
74=item * B<count>
75
76Returns the number of elements in the array.
77
78 $stuff = Stuff->new;
79 $stuff->options( [ "foo", "bar", "baz", "boo" ] );
80
81 print $stuff->count_options; # prints 4
82
83This method does not accept any arguments.
84
85=item * B<is_empty>
86
87Returns a boolean value that is true when the array has no elements.
88
89 $stuff->has_no_options ? die "No options!\n" : print "Good boy.\n";
90
91This method does not accept any arguments.
92
93=item * B<elements>
94
95Returns all of the elements of the array as an array (not an array reference).
96
97 my @option = $stuff->all_options;
98 print "@options\n"; # prints "foo bar baz boo"
99
100This method does not accept any arguments.
101
102=item * B<get($index)>
103
104Returns an element of the array by its index. You can also use negative index
105numbers, just as with Perl's core array handling.
106
107 my $option = $stuff->get_option(1);
108 print "$option\n"; # prints "bar"
109
110If the specified element does not exist, this will return C<undef>.
111
112This method accepts just one argument.
113
114=item * B<pop>
115
116Just like Perl's builtin C<pop>.
117
118This method does not accept any arguments.
119
120=item * B<push($value1, $value2, value3 ...)>
121
122Just like Perl's builtin C<push>. Returns the number of elements in the new
123array.
124
125This method accepts any number of arguments.
126
127=item * B<shift>
128
129Just like Perl's builtin C<shift>.
130
131This method does not accept any arguments.
132
133=item * B<unshift($value1, $value2, value3 ...)>
134
135Just like Perl's builtin C<unshift>. Returns the number of elements in the new
136array.
137
138This method accepts any number of arguments.
139
140=item * B<splice($offset, $length, @values)>
141
142Just like Perl's builtin C<splice>. In scalar context, this returns the last
143element removed, or C<undef> if no elements were removed. In list context,
144this returns all the elements removed from the array.
145
146This method requires at least one argument.
147
148=item * B<first( sub { ... } )>
149
150This method returns the first matching item in the array, just like
151L<List::Util>'s C<first> function. The matching is done with a subroutine
152reference you pass to this method. The subroutine will be called against each
153element in the array until one matches or all elements have been checked.
154
155 my $found = $stuff->find_option( sub {/^b/} );
156 print "$found\n"; # prints "bar"
157
158This method requires a single argument.
159
160=item * B<first_index( sub { ... } )>
161
162This method returns the index of the first matching item in the array, just
163like L<List::MoreUtils>'s C<first_index> function. The matching is done with a
164subroutine reference you pass to this method. The subroutine will be called
165against each element in the array until one matches or all elements have been
166checked.
167
168This method requires a single argument.
169
170=item * B<grep( sub { ... } )>
171
172This method returns every element matching a given criteria, just like Perl's
173core C<grep> function. This method requires a subroutine which implements the
174matching logic.
175
176 my @found = $stuff->filter_options( sub {/^b/} );
177 print "@found\n"; # prints "bar baz boo"
178
179This method requires a single argument.
180
181=item * B<map( sub { ... } )>
182
183This method transforms every element in the array and returns a new array,
184just like Perl's core C<map> function. This method requires a subroutine which
185implements the transformation.
186
187 my @mod_options = $stuff->map_options( sub { $_ . "-tag" } );
188 print "@mod_options\n"; # prints "foo-tag bar-tag baz-tag boo-tag"
189
190This method requires a single argument.
191
192=item * B<reduce( sub { ... } )>
193
194This method turns an array into a single value, by passing a function the
195value so far and the next value in the array, just like L<List::Util>'s
196C<reduce> function. The reducing is done with a subroutine reference you pass
197to this method.
198
199 my $found = $stuff->reduce_options( sub { $_[0] . $_[1] } );
200 print "$found\n"; # prints "foobarbazboo"
201
202This method requires a single argument.
203
204=item * B<sort>
205
206=item * B<sort( sub { ... } )>
207
208Returns the elements of the array in sorted order.
209
210You can provide an optional subroutine reference to sort with (as you can with
211Perl's core C<sort> function). However, instead of using C<$a> and C<$b> in
212this subroutine, you will need to use C<$_[0]> and C<$_[1]>.
213
214 # ascending ASCIIbetical
215 my @sorted = $stuff->sort_options();
216
217 # Descending alphabetical order
218 my @sorted_options = $stuff->sort_options( sub { lc $_[1] cmp lc $_[0] } );
219 print "@sorted_options\n"; # prints "foo boo baz bar"
220
221This method accepts a single argument.
222
223=item * B<sort_in_place>
224
225=item * B<sort_in_place( sub { ... } )>
226
227Sorts the array I<in place>, modifying the value of the attribute.
228
229You can provide an optional subroutine reference to sort with (as you can with
230Perl's core C<sort> function). However, instead of using C<$a> and C<$b>, you
231will need to use C<$_[0]> and C<$_[1]> instead.
232
233This method does not define a return value.
234
235This method accepts a single argument.
236
237=item * B<shuffle>
238
239Returns the elements of the array in random order, like C<shuffle> from
240L<List::Util>.
241
242This method does not accept any arguments.
243
244=item * B<uniq>
245
246Returns the array with all duplicate elements removed, like C<uniq> from
247L<List::MoreUtils>.
248
249This method does not accept any arguments.
250
251=item * B<join($str)>
252
253Joins every element of the array using the separator given as argument, just
254like Perl's core C<join> function.
255
256 my $joined = $stuff->join_options(':');
257 print "$joined\n"; # prints "foo:bar:baz:boo"
258
259This method requires a single argument.
260
261=item * B<set($index, $value)>
262
263Given an index and a value, sets the specified array element's value.
264
265This method returns the value at C<$index> after the set.
266
267This method requires two arguments.
268
269=item * B<delete($index)>
270
271Removes the element at the given index from the array.
272
273This method returns the deleted value. Note that if no value exists, it will
274return C<undef>.
275
276This method requires one argument.
277
278=item * B<insert($index, $value)>
279
280Inserts a new element into the array at the given index.
281
282This method returns the new value at C<$index>.
283
284This method requires two arguments.
285
286=item * B<clear>
287
288Empties the entire array, like C<@array = ()>.
289
290This method does not define a return value.
291
292This method does not accept any arguments.
293
294=item * B<accessor($index)>
295
296=item * B<accessor($index, $value)>
297
298This method provides a get/set accessor for the array, based on array indexes.
299If passed one argument, it returns the value at the specified index. If
300passed two arguments, it sets the value of the specified index.
301
302When called as a setter, this method returns the new value at C<$index>.
303
304This method accepts one or two arguments.
305
306=item * B<natatime($n)>
307
308=item * B<natatime($n, $code)>
309
310This method returns an iterator which, on each call, returns C<$n> more items
311from the array, in order, like C<natatime> from L<List::MoreUtils>. A coderef
312can optionally be provided; it will be called on each group of C<$n> elements
313in the array.
314
315This method accepts one or two arguments.
316
317=item * B<shallow_clone>
318
319This method returns a shallow clone of the array reference. The return value
320is a reference to a new array with the same elements. It is I<shallow>
321because any elements that were references in the original will be the I<same>
322references in the clone.
323
324=back
